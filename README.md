# mtl-core (多端化能力SDK)

## 关键特性

- 设备访问能力：定位、打电话、打开相机、录音等。（summer.js 默认包含）
- 多端协同能力：投票等
- 智能化能力：VUI等

## 如何使用

下载：
```
$ npm i mtl-core -S
```

项目中引入：
```
import xx from 'mtl-core/xx';

```
